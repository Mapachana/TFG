Ayuda

Preguntas frecuentes

  • ¿Qué es este sitio?
  • Una página web con conceptos introductorios a diversos modelos de epidemiología y ajuste de parámetros.
  • ¿Dónde puedo ampliar información?
  • Entre diversas fuentes, un libro muy útil es "mathematical_models_in_population_biology_and_epidemiology"
  • ¿Dónde puedo consultar los conceptos necesarios presentados?
  • Un libro con las matemáticas necesarias es "Discrete Dynamical Models"
  • ¿Qué es la notación?
  • La notación es el sistema de signos o símbolos que se utiliza para representar ciertos conceptos.
  • No se me renderizan las gráficas interactivas, ¿qué hago?
  • Refresca la página o espera un minuto, a veces tardan en cargar.
  • ¿Qué formato de archivo tengo que subir para realizar ajustes de datos?
  • Se debe subir un .csv con los datos de susceptibles, infectados y recuperados (si lo hubiera).
